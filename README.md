빌드 산출물(Windows exe/msi)과 자동 업데이트 매니페스트(`latest.json`)만 배포하는 저장소입니다.
소스코드는 여기 없습니다.

- 다운로드: [Releases](https://github.com/crecon31/invoice-builder-releases/releases) 탭에서 최신 exe/msi 받으세요.
- 자동 업데이트: 앱이 이 저장소의 `releases/latest/download/latest.json`을 주기적으로 확인합니다.
